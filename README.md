# AISoccerTeamMaker
⚽ AI Squad Maker divide un gruppo di giocatori in due squadre equilibrate. Analizza i dati storici (voti, presenze, performance) con un modello ML locale per calcolare la forza di ogni giocatore, poi trova la divisione ottimale bilanciando livello e ruoli. Tutto gira in locale, nessun dato inviato online.
